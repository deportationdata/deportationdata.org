## Usage Guidelines

1. **Consistency**: Use only the standardized terms from this table
2. **Context**: Some terms like "deportación" vs "expulsión" depend on legal context
3. **Updates**: Verify with government sources when new terms are introduced
4. **Documentation**: Always cite government sources for new technical translations
